# Business Sales Analytics Dashboard

## Project Overview

This project focuses on analyzing business sales data to identify sales trends, profitable regions, and top-performing products using Python and Tableau.

The project helps in understanding business performance through data visualization and analytical insights.


## Technologies Used

* Python
* Pandas
* Tableau Public


## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Sales and Profit Analysis
4. Category-wise Analysis
5. Region-wise Analysis
6. Dashboard Creation using Tableau


## Analysis Performed

* Monthly Sales Trend Analysis
* Category-wise Sales Analysis
* Region-wise Profit Analysis
* Top 10 Products Analysis


## Dashboard Components

1. Monthly Sales Trend

Visualizes monthly sales performance over time.

2. Category Sales

Displays sales distribution across product categories.

3. Region Profit

Shows profit generated from different business regions.

4. Top Products

Highlights the top-performing products based on sales.


## Project Files

* `superstore.csv` → Dataset used for analysis
* `sales_analysis.py` → Python analysis script
* `tableau_dashboard.png` → Tableau dashboard screenshot


## Outcome

Successfully created an interactive business sales dashboard and generated meaningful insights from sales data using data analytics and visualization techniques.
